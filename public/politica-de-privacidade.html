Política de Privacidade

O Sabor Global respeita a sua privacidade. Esta página informa sobre nossas políticas em relação à coleta, uso e divulgação de dados.

Coleta e Uso de Dados

Nós não coletamos informações de identificação pessoal diretamente. No entanto, usamos tecnologias e serviços de terceiros que podem coletar dados:

  • LocalStorage: Utilizamos o LocalStorage do seu navegador para salvar suas "Receitas Favoritas". Esses dados são armazenados exclusivamente no seu dispositivo e não são enviados para nossos servidores.
  • Google AdSense: Este site exibirá anúncios do Google AdSense. O Google usa cookies para veicular anúncios com base nas visitas anteriores do usuário ao seu site ou a outros sites. Você pode desativar a publicidade personalizada visitando as Configurações de anúncios do Google.
  • APIs de Terceiros: Nosso serviço utiliza a API da Edamam para buscar receitas e a API do Google Translate para traduções. O uso dessas APIs está sujeito às suas respectivas políticas de privacidade.

Links para Outros Sites

Nosso serviço contém links para outros sites que não são operados por nós. Se você clicar em um link de terceiros, será direcionado para o site desse terceiro. Aconselhamos vivamente que reveja a Política de Privacidade de todos os sites que visitar.
